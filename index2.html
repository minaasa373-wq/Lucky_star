1. いま何が起きてる？

AI動画生成は、数ヶ月単位で主役が入れ替わる超高速の進化期に入っている。大事なのは、ランキングを見ることより、「何を作るか」に応じて使い分けること。

2. モデルは“用途”で選ぶ

画質差だけを見る時代は終わりつつある。差が出るのは、動き、物理表現、音、量産性、実務とのつなぎやすさ。

こんな人はSeedance向き
  • 実写CMや映画っぽい映像を作りたい
  • 髪、服、重み、光の自然さを重視したい
  • 短いけど濃い1本を作りたい
こんな人はVeo向き
  • 映像だけでなく音も一緒に考えたい
  • 効果音、環境音、しゃべりもまとめて試したい
  • Google系の流れで使いたい
こんな人はKling向き
  • 複数カットの構成を一気に試したい
  • コストを抑えながら試作したい
  • ショート動画のバリエーションを増やしたい
判断のコツ

先に「作りたい動画の勝ち筋」を決める。
例：実写CM、アバター説明、縦型広告、商品紹介、映画風、SNS量産。

3. HeyGenとAdobe Fireflyが意味すること

最新トレンドは“生成だけ”ではなく、“制作フローにどう組み込むか”。ここが実務でかなり大事。

HeyGen

映画風に動くアバターを作りやすい。最大3体まで同じシーンに置けるため、対話動画やプレゼン動画にも使いやすい。

弱点

地域制限、安定性、対象国の制約などがある場合がある。万能ではなく、様子見しながら使う道具。

Adobe Firefly

外部AIモデルをAdobe環境で切り替えやすくなってきた。生成→ダウンロード→編集の手間を減らしやすい。

Premiere連携

編集ソフトへ持って行くまでの摩擦が減る。制作スピードは上がるが、クレジット管理はより重要になる。

データ送信

外部モデル統合では、プロンプトや素材が他社処理になる場合がある。社外秘や機密は特に注意。

実務の目線

“作れるか”だけでなく、“社内ルールに乗るか”“クライアント確認がいるか”も見る。

4. AI動画生成でハマりやすい7つの罠

初心者が失敗する原因は、センス不足ではない。設計不足と確認不足がほとんど。

一番多いのは、プロンプトを盛りすぎてAIが混乱することです。顔、手、背景、カメラ、感情、光、全部を一度に詰め込むと壊れやすいです。

柊かがみ解説役

だから、動かす対象を絞るの。激しい動きなら1つ、中くらいなら2つ、小さい動きなら3つくらいまでが目安。あと、原型画像を作ってから動画にするのが本当に大事。

やさしく言うと：最初から全部入りラーメンにしない。まずは麺とスープを安定させる。
罠1：盛りすぎ

情報を詰め込みすぎると破綻しやすい。動きと対象を絞る。

罠2：雑すぎ

短すぎる指示だとAIが勝手に補完する。主体、場所、動き、カメラ、雰囲気は入れる。

罠3：原型なし

静止画を先に作らず動画へ行くと、顔、服、背景の一貫性が崩れやすい。

罠4：道具ミス

実務向け、アニメ向け、映画向け、アバター向けでツールは違う。

罠5：生成ガチャ

回し続けるとクレジットが消える。3回ルールで止める。

罠6・7：確認不足と権利軽視

数字、日本語、ロゴ、商用条件、既存キャラ、有名人、肖像権を軽く見ると危険。

5. AIラベル・法律・やってはいけないこと

これからは“作れるか”だけでなく、“出していいか”が超重要。技術より先に、ここを知らないと危ない。

YouTubeのAIラベル

YouTube側がAI生成コンテンツを検知した場合、自動でラベルが付く流れが強まっている。AIラベル自体が即悪ではないが、隠す・誤魔化す・申告しない方が危ない。

人間チェックが必須

人が編集した動画にAIラベルが付く可能性もあり得る。だからこそ、制作ログ、素材の出どころ、使ったツールを整理しておくと安心。

実在人物の危険な加工

特に未成年や実在人物を使った不適切なAI加工は絶対に避ける。作っただけ、持っているだけでも問題になることがある。

覚えておくべき原則

「AIで作ったから安全」は通用しない。著作権、肖像権、商用利用規約、プラットフォーム規約は必ず見る。

超重要：
実在人物、とくに未成年、既存キャラ、有名人、他社ブランドのロゴや商品、無料素材の商用条件には要注意。 AI時代は“何を作れるか”より、“何を作ってはいけないか”を知っている人が強い。

6. 新しい時代に対応できる方法

AI動画制作は、技術だけ磨いても足りない。運用、確認、倫理までセットで回せる人が、これから信用される。

対応法1：用途から逆算する

“何の動画か”を先に決める。SNSショート、商品紹介、研修、CM、映画風で必要な道具は変わる。

対応法2：静止画から始める

顔、服、背景、色味、距離感を静止画で固めてから動かす。これだけで事故率がかなり下がる。

対応法3：短く具体的なプロンプト

誰が、どこで、何をして、カメラはどう動き、どんな雰囲気か。この順で十分強い。

対応法4：3回で切る

1回目=理想、2回目=安全版、3回目=少し攻める。4回目以降は別アプローチに切り替える。

対応法5：公開前の点検表

数字、日本語、名称、ロゴ、権利、AIラベル、音声、字幕、事実確認。これを毎回見る。

対応法6：成功パターンを残す

良かったプロンプト、ダメだった設定、使ったモデル、所感をメモして、再現しやすい型にする。

2. 精度を上げるプロンプトは「細かく」ではなく「明確に」書く

細かく書けばいいわけではない。大事なのは、AIが迷うところを先回りして埋めること。世界観、表情、トーン、動き、カメラ、禁止事項を整理して書く。

弱い例

「女性が商品を紹介する動画」
→ 場所、表情、カメラ、商品、雰囲気、音声、動きが不明。

強い例

「明るい朝のキッチン。30代女性が白いテーブルの前で商品を手に持ち、安心した表情でゆっくり説明する。カメラは固定、ズームなし。背景は動かさない。口元は日本語音声に合わせる。手を増やさない。」

世界観

時代、場所、空気感、光、天候を書く。映画・MV・ファンタジーでは特に重要。

表情

不安、希望、緊張、疲れ、安心などを指定する。演出と顔のズレを減らす。

トーン

CM風、Vlog風、映画風、ドキュメンタリー風、SF風、アニメ風などを入れる。

3. 動きとカメラは、欲張るほど壊れやすい

動画生成AIは、激しい動き、複数人、手、口、カメラ移動が重なると破綻しやすい。最初は小さく動かし、成功したら少しずつ増やす。

動きは小さく

控えめ、ゆっくり、一定速度で、軽く振り返る、髪が少し揺れる、などから始める。

画角をロック

カメラ固定、ズームなし、パンなし、背景固定を入れると、勝手な移動を防ぎやすい。

人数を減らす

人物が多いほど破綻しやすい。不要なら1〜2人に抑える。

速度を指定

高速、ゆっくり、一定速度、急加速しない、などで躍動感と安定性を調整する。

物理法則

月面なら浮く、物は自然に落下する、服と髪は重力に従うなどを入れる。

POV

一人称視点、動物視点、ダイバー視点、パイロット視点などで臨場感を出す。

4. リファレンス・リップシンク・開始終了画像を使う

キャラクター固定や商品レビューのように、一貫性が大事な動画では、テキストだけで頑張るより、画像や音声を使った方が安定する。

リファレンス

参照画像を使うと、同じ人物、同じ服装、同じアクセサリーを保ちやすい。

開始画像と終了画像

朝から夜、室内から屋外など、変化のある動画は始点と終点を指定すると安定しやすい。

リップシンク

音声と口の動きを合わせる。MV、会話、レビュー、Vlog風動画では重要。

モーションコントロール

ダンスやアクションは、動きの参照を使うと再現しやすい。

背景の展開

トンネルに入る、景色が変わる、雨が降るなどを入れると単調さを避けられる。

動画の続きを作る

最後のフレームを使って続きを生成すると、短い動画を自然に長くしやすい。

5. 生成後チェックで信用を守る

高性能モデルでも、同じプロンプトから完全に同じ動画は出ない。だから、成功プロンプトの丸コピーより、破綻しにくい構造を理解する方が重要。

必ず見る場所
  • 顔、手、指、口、目線
  • 日本語音声、字幕、看板、ロゴ
  • 数字、年号、統計、名称
  • 商品や人物の一貫性
  • 商用利用、著作権、肖像権
3回ルール

1回目は理想形、2回目は安全版、3回目は少し攻める。3回やってダメなら、モデルを変えるか、原型画像やプロンプトを作り直す。

結論：
AI動画生成は、最初から明確に指示すると、修正回数とクレジット消費を減らせる。ただし最後に信用を守るのは人間。AIに作らせ、人間が確認し、必要なら編集で直す。

1. 音量の黄金バランスを先に覚える

SCENE 31 / 音量の基本
柊つかさ素朴な疑問役

目安は -28〜-25dB前後。雰囲気づくり担当。声を押しつぶさない位置に置く。

SE

目安は -15〜-20dB前後。場面のアクセント。高い音は刺さりやすいので要注意。

声の設定例

リミッター（上限）-5dB前後。コンプレッサー（ばらつき抑制）は補正10〜15程度。声の大小差を減らす。

SEの設定例

リミッター -15dB前後。コンプレッサーは5程度。キラキラ音、チリン音など刺さる音だけは個別で少し下げる。

BGMの設定例

リミッター -25〜-28dB程度。コンプレッサーは軽め。声を邪魔しない土台にする。

リミッター

音量の上限を決める機能。これ以上大きくならない壁を作るイメージ。

コンプレッサー

小さい音を少し持ち上げ、大きい音を抑えて、ばらつきを減らす機能。

ダイナミック

上の2つをまとめて考えるイメージでOK。まずは音量差を整えるために使う。

3. ノイズ除去は「必要な時だけ、控えめに」

エアコンのサー音、家電のブーン音、部屋のホワイトノイズは、軽く掃除するだけでもかなり聞きやすくなる。ただし強すぎると音がこもる。

ホワイトノイズ

サーっと常に鳴る雑音。エアコン、PCファン、家電音などが原因になりやすい。

Premiereでの考え方

声トラックや問題のあるトラックにノイズ除去を控えめに適用。10〜20程度の弱めから試す。

やりすぎ注意

除去を強くすると、声まで削ってこもりやすい。ゼロにしようとしすぎない方が自然。

AIツール活用

ユニコンバーター系ツールは、AIで自動ノイズ除去しやすい。録音の掃除を手早くやりたい人向き。

ボーカル分離

① BGMの使いたい部分を決める → ② キーフレームで少しずつ音量を下げる → ③ スタジオリバーブを入れる → ④ 前後をクロスフェードでつなぐ。

Premiere Proのスタジオリバーブ。プリセットは「大ホール」から始めると分かりやすい。かけすぎは禁物。

もっと簡単に：音に余韻をつけるエフェクト。
クロスフェード
意味：前の音を少しずつ消しながら、次の音を少しずつ出すつなぎ方。
もっと簡単に：音の切り替えをなめらかにする方法。
ノイズ除去
意味：不要な雑音を減らす処理。
もっと簡単に：サー音やブーン音を目立たなくすること。
リップシンク
意味：音声と口の動きを合わせる技術。
もっと簡単に：しゃべる声に合わせて口パクさせること。
BGM
意味：Background Music。映像の後ろで流す音楽。
もっと簡単に：動画の雰囲気を作る背景音楽。
商用利用
意味：収益や仕事につながる用途で使うこと。
もっと簡単に：YouTube収益化や仕事動画で使うこと。
クレジット表記
意味：作者名、曲名、配布元などを表示すること。
もっと簡単に：概要欄に「この曲を使いました」と書くこと。
BPM
意味：1分間の拍数。曲の速さを表す数字。
もっと簡単に：数字が大きいほど速い曲。
フェードイン / フェードアウト
意味：音や映像を少しずつ出したり消したりする編集。
もっと簡単に：急に始まる・急に終わる感じをやわらげる方法。
インスト
意味：歌が入っていない楽器中心の曲。
もっと簡単に：声がないBGM。
HDD
意味：大容量を安く保存しやすい記憶装置。
もっと簡単に：安くたくさん入る保存箱。完成動画の保管向き。
SSD
意味：高速で衝撃にも強い記憶装置。
もっと簡単に：速い保存箱。編集中の作業向き。
クラウド
意味：インターネット上に保存する仕組み。
もっと簡単に：ネットの倉庫。共有やバックアップに便利。
バックアップ
意味：データが消えた時に備えて別の場所へ複製しておくこと。
もっと簡単に：予備コピーを作ること。
Lumetriカラー
意味：Premiere Proの色補正機能。
もっと簡単に：映像の明るさや色味を整える道具。
マスク
意味：見せる部分と隠す部分を指定する機能。
もっと簡単に：人物だけ残して背景を変えるための切り抜き指定。
ネスト
意味：複数クリップを1つのまとまりとして扱う機能。
もっと簡単に：細かいパーツを1つにまとめる箱。
FPS
意味：1秒間に何枚の映像を見せるかの数。
もっと簡単に：動画のコマ数。24FPSは映画っぽい見え方の基本。
AE/AFロック
意味：明るさ（AE）とピント（AF）を固定する機能。
もっと簡単に：スマホが勝手に明るさやピントを変えないように止める機能。
オービットショット
意味：被写体の周囲を回りながら撮る方法。
もっと簡単に：主役のまわりをぐるっと回る撮り方。
Vlog
意味：Video Blog。日常や体験を動画で記録・発信する形式。
もっと簡単に：動画の日記みたいなもの。
前ボケ
意味：カメラの手前にある物をぼかす表現。
もっと簡単に：手前をぼかして主役を引き立てる撮り方。
玉ボケ
意味：光が丸くぼけて見える表現。
もっと簡単に：イルミネーションの丸いキラキラみたいな見え方。
リファレンス
意味：AIに見本として渡す参照画像や参照素材。
もっと簡単に：「この顔・服・雰囲気を保って」と見せる見本。
リップシンク
意味：音声に合わせて口の動きを合わせること。
もっと簡単に：しゃべっている声と口パクを合わせる機能。
POV
意味：Point of View。一人称視点。
もっと簡単に：自分の目で見ているようなカメラ視点。
モーションコントロール
意味：参照動画や指示を使って、動き方を制御する方法。
もっと簡単に：AIに「この動きをまねして」と教えること。
マルチモーダル
意味：文字、画像、音声、動画など、複数の種類の情報をまとめて扱う仕組み。
もっと簡単に：文章だけじゃなく、絵も音も一緒にわかるAI。
API
意味：外部サービス同士をつなぐための仕組み。
もっと簡単に：ソフト同士の連絡口。
物理表現
意味：布、髪、重力、光、動き方などが自然に見える表現。
もっと簡単に：“ちゃんと現実っぽく動いて見えるか”のこと。
AIラベル
意味：AIで作られた、またはAI加工が入ったコンテンツであることを示す表示。
もっと簡単に：「この動画、AI使ってます」の名札。
クレジット
意味：AI生成に使うポイントや回数のこと。
もっと簡単に：生成すると減る“体力ゲージ”。
アバター
意味：自分やキャラクターの代わりに動画へ登場する分身。
もっと簡単に：顔出しの代わりにしゃべってくれる代理人。
著作権・肖像権
意味：作品や人物の顔・姿を勝手に使われないための権利。
もっと簡単に：他人の作品や顔を、許可なく使っちゃダメというルール。
